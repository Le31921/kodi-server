## Installation

To install the application, follow these steps:

1. Make sure you have [Node.js](https://nodejs.org/en/download/current) installed.
2. Navigate to the project directory:
  ```bash
  cd kodi-server
  ```
3. Install dependencies
```bash
npm i
```
